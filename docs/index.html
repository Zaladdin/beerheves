Как использовать этот репозиторий

1. GitHub

  1. Создайте новый репозиторий.
  2. Загрузите проект в main.
  3. При необходимости создайте файл .env на сервере из .env.example.

2. GitHub Pages

  1. Включите Pages для репозитория.
  2. Используйте GitHub Actions как source.
  3. На Pages будет открываться эта статическая preview-страница из docs/.

3. Реальный backend deploy

  1. Размещайте PHP-часть на хостинге с PHP 8.2+ и MySQL.
  2. Document root должен смотреть в public/.
  3. Импортируйте database/schema.sql и database/seed.sql.
